# Excel Sales Dashboard

## Project Overview

This project analyzes sales data using Microsoft Excel.

The dashboard was built using:
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Bar Charts
- Pie Charts
- KPI Metrics
- Interactive Dashboard Design

## Sheets Included

### 📋 Raw Data
Contains the original sales dataset.

### 📊 Dashboard
Main dashboard showing business KPIs and insights.

### 📈 State Charts
State-wise sales performance analysis.

### 🥧 Category & Segment Analysis
Category and customer segment breakdown.
